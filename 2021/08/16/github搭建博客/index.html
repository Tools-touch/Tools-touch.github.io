github搭建博客

window上安装

一、软件与环境：

  • nodejs : (安装最新版本)

    https://mirrors.aliyun.com/nodejs-release/

  • git

    https://git-scm.com/download/win

  • 安装cnpm

查看cnpm版本

  • 安装hexo
  • 查看hexo版本

二、创建博客

  • 创建一个文件夹
  • 进入文件夹右键点击 Git Bash Here
  • 创建博客文件 hexo init
  • 在本地查看博客 hexo s

三、部署到GitHub上面

  • 在GitHub上创建空间
  • [创建仓库][https://raw.githubusercontent.com/Tools-touch/img/main/%E5%88%9B%E5%BB%BA%E4%BB%93%E5%BA%93.png]
  • [仓库命名][https://raw.githubusercontent.com/Tools-touch/img/main/%E4%BB%93%E5%BA%93%E5%91%BD%E5%90%8D.png]
  • 在git面板输入
  • 修改 __config.yml 文件
  • 在 git 面板设置 GitHub 用户邮箱
  • 部署上去 hexo d
  • 访问博客 用户名.github.io

四、解析到域名上

  • 在仓库上创建一个 CNAME 文件
  • CNAME 文件里填写要绑定的域名且不要包含 Http://和www
  • 到域名商将域名添加记录值 CANME 解析到 用户名.github.io
